# Immich Backup Manager 1.3.2

## English

Version 1.3.2 protects the locally stored Immich API key with Windows DPAPI. The encrypted value is bound to the current Windows user account and cannot normally be decrypted by another Windows account. Existing plaintext settings created by version 1.3.1 are automatically migrated on first start.

## Deutsch

Version 1.3.2 schützt den lokal gespeicherten Immich-API-Key mit Windows DPAPI. Der verschlüsselte Wert ist an das aktuelle Windows-Benutzerkonto gebunden und kann normalerweise nicht von einem anderen Windows-Konto entschlüsselt werden. Vorhandene Klartext-Einstellungen aus Version 1.3.1 werden beim ersten Start automatisch migriert.
