Release title:
--------------
Immich Backup Manager 1.3.2

Tag:
----
v1.3.2

Release description:
--------------------
English:
Version 1.3.2 encrypts the locally stored Immich API key using Windows DPAPI. The encrypted value is bound to the current Windows user account. Existing plaintext settings from version 1.3.1 are automatically migrated on first start. The German and English information dialog has also been expanded with details about encryption, storage location, privacy, migration, and deletion of saved credentials.

Deutsch:
Version 1.3.2 verschlüsselt den lokal gespeicherten Immich-API-Key mit Windows DPAPI. Der verschlüsselte Wert ist an das aktuelle Windows-Benutzerkonto gebunden. Vorhandene Klartext-Einstellungen aus Version 1.3.1 werden beim ersten Start automatisch migriert. Zusätzlich wurde der deutsche und englische Info-Dialog um Angaben zu Verschlüsselung, Speicherort, Datenschutz, Migration und Löschung gespeicherter Zugangsdaten erweitert.
