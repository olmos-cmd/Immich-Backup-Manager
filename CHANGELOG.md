# Changelog

# English

## Version 1.3.0

- Added Dark Mode with light/dark theme switching
- Added complete German and English user interface
- Redesigned album cards and preview areas
- Modernized the interface with turquoise accent colors
- Improved download progress display and usability
- Added scrollable settings and information panels
- Expanded freeware, copyright, privacy and disclaimer information
- Fixed numerous Rust, user-interface and build issues
- Successfully verified the Windows build
- No known outstanding issues

## Version 1.2.0

- Added downloads for Immich albums and photos grouped by year
- Added support for personal and shared albums
- Added backup folder selection and handling of existing files
- Added parallel downloads with progress display
- Added local API-key storage and deletion
- Added initial freeware, copyright and privacy information
