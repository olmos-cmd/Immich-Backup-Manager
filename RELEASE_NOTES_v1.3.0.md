# Immich Backup Manager v1.3.0

A major update with a redesigned interface, Dark Mode, complete German and English support, improved album cards and a clearer download workflow.

## English

### Highlights

- Dark Mode with light/dark theme switching
- Complete German and English user interface
- Redesigned album cards and preview areas
- Modern turquoise accent colors
- Improved progress display and usability
- Scrollable settings and information panels
- Expanded freeware, copyright, privacy and disclaimer information
- Numerous Rust, UI and build fixes
- Windows build successfully verified
- No known outstanding issues

### Download

Download the Windows package from the **Assets** section of the GitHub release.
