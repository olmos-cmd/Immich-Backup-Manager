# Immich Backup Manager 1.3.1

## Deutsch

Version 1.3.1 verbessert die Darstellung der Duplikatverwaltung. Das Vergleichsfenster wurde verkleinert, die beiden Vergleichsbereiche und Bildvorschauen wurden kompakter gestaltet und die unteren Aktionsschaltflächen bleiben nun auch bei geringerer Bildschirmhöhe sichtbar.

### Änderungen

- Duplikatfenster verkleinert und optimiert
- Vergleichsboxen kompakter gestaltet
- Bildvorschauen verkleinert
- Untere Aktionsschaltflächen dauerhaft sichtbar
- Weitere Layout- und Darstellungsverbesserungen
- Versionsnummer auf 1.3.1 erhöht

### Installation

1. ZIP-Datei herunterladen und entpacken.
2. `Immich Backup Manager.exe` starten.
3. Bei einer vorhandenen Installation kann die bisherige EXE durch die neue Version ersetzt werden.

## English

Version 1.3.1 improves the duplicate-management interface. The comparison window has been reduced in size, both comparison panels and image previews are more compact, and the lower action buttons now remain visible on displays with less available height.

### Changes

- Reduced and optimized the duplicate-management window
- Made comparison panels more compact
- Reduced image-preview sizes
- Ensured that lower action buttons remain visible
- Added further layout and display improvements
- Increased the version number to 1.3.1

### Installation

1. Download and extract the ZIP archive.
2. Start `Immich Backup Manager.exe`.
3. For an existing installation, replace the previous EXE with the new version.
