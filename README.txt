IMMICH BACKUP MANAGER 1.3.0

English
-------
Immich Backup Manager is a Windows application for downloading original photos
and videos from an Immich server by album or year.

Main features:
- Personal and shared albums
- Photos without albums grouped by year
- All photos grouped by year
- Parallel downloads
- Existing-file comparison
- Light and Dark Mode
- German and English interface
- Local API-key storage and deletion

Deutsch
-------
Immich Backup Manager ist ein Windows-Programm zum Herunterladen von
Originalfotos und Originalvideos aus einer Immich-Installation nach Album
oder Jahr.

Hauptfunktionen:
- Eigene und geteilte Alben
- Fotos ohne Album nach Jahr
- Alle Fotos nach Jahr
- Parallele Downloads
- Vergleich vorhandener Dateien
- Hell- und Dunkelmodus
- Deutsche und englische Oberfläche
- Lokale Speicherung und Löschung des API-Keys
